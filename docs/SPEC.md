# GrowthOS 功能规格文档

## 1. 概述

本文档详细描述 GrowthOS 系统的功能需求和规格定义，作为产品设计和开发的基础参考。

---

## 2. 用户角色

| 角色 | 描述 | 权限 |
|------|------|------|
| 访客 | 未登录用户 | 浏览登录/注册页面 |
| 注册用户 | 完成注册的用户 | 使用所有核心功能 |

## 3. 功能模块

### 3.1 认证模块

#### 3.1.1 用户注册

| 项目 | 规格 |
|------|------|
| 功能描述 | 新用户创建账户 |
| 触发方式 | 注册页面表单提交 |
| 前置条件 | 无 |

**输入规格：**

| 字段 | 类型 | 必填 | 验证规则 |
|------|------|------|----------|
| 邮箱 | string | 是 | 有效邮箱格式 |
| 密码 | string | 是 | 最少6位 |
| 确认密码 | string | 是 | 与密码一致 |

**业务规则：**
- 邮箱不能与已注册用户重复
- 密码需加密存储（SHA-256）
- 注册成功后自动登录

**输出规格：**
- 成功：跳转首页，显示欢迎提示
- 失败：显示错误信息

#### 3.1.2 用户登录

| 项目 | 规格 |
|------|------|
| 功能描述 | 用户登录系统 |
| 触发方式 | 登录页面表单提交 |
| 前置条件 | 账户已注册 |

**输入规格：**

| 字段 | 类型 | 必填 | 验证规则 |
|------|------|------|----------|
| 邮箱 | string | 是 | 有效邮箱格式 |
| 密码 | string | 是 | 不为空 |

**业务规则：**
- 验证邮箱和密码匹配
- 登录成功后存储会话
- 登录失败次数限制（可选）

**输出规格：**
- 成功：跳转首页
- 失败：显示错误信息

#### 3.1.3 用户登出

| 项目 | 规格 |
|------|------|
| 功能描述 | 用户退出登录 |
| 触发方式 | 点击登出按钮 |
| 前置条件 | 用户已登录 |

**业务规则：**
- 清除本地会话
- 跳转登录页面

#### 3.4.2 创建记录

| 项目 | 规格 |
|------|------|
| 功能描述 | 创建新的每日记录 |
| 触发方式 | 点击添加按钮 |

**输入规格：**

| 字段 | 类型 | 必填 | 验证规则 |
|------|------|------|----------|

- 每天只能创建一条记录

#### 3.5.1 目标列表

| 触发方式 | 访问目标页面 |
| 前置条件 | 用户已登录 |

**功能规格：**
- 按状态分组显示（进行中/已完成/已取消）
- 显示进度百分比
- 支持创建新目标

#### 3.5.2 创建目标

| 项目 | 规格 |
|------|------|
| 功能描述 | 创建新的目标 |
| 触发方式 | 点击创建按钮 |
| 前置条件 | 用户已登录 |

**输入规格：**

| 字段 | 类型 | 必填 | 验证规则 |
|------|------|------|----------|
| 标题 | string | 是 | 1-100字符 |
| 描述 | string | 否 | 最多500字符 |
| 目标值 | number | 是 | 1-1000 |
| 开始日期 | date | 是 | 不早于今天 |
| 结束日期 | date | 是 | 晚于开始日期 |

**业务规则：**
- 结束日期必须晚于开始日期
- 目标值必须大于0

#### 3.5.3 更新目标进度

| 项目 | 规格 |
|------|------|
| 功能描述 | 更新目标当前进度 |
| 触发方式 | 滑动进度条或输入数值 |
| 前置条件 | 用户已登录且目标存在 |

**输入规格：**

| 字段 | 类型 | 必填 | 验证规则 |
|------|------|------|----------|
| 当前值 | number | 是 | 0-目标值 |

**业务规则：**
- 当前值不能超过目标值
- 当前值达到目标值时自动标记为完成

#### 3.5.4 目标状态管理

| 项目 | 规格 |
|------|------|
| 功能描述 | 变更目标状态 |
| 触发方式 | 选择状态 |
| 前置条件 | 用户已登录且目标存在 |

**状态选项：**
- 进行中：目标正在执行
- 已完成：目标已达成
- 已取消：目标被取消

### 3.7 数据分析模块

#### 3.7.1 统计数据

| 项目 | 规格 |
|------|------|
| 功能描述 | 展示成长数据统计 |
| 触发方式 | 访问分析页面 |
| 前置条件 | 用户已登录 |

**统计指标：**

| 指标 | 描述 | 计算方式 |
|------|------|----------|

| 目标完成率 | 已完成目标占比 | completed/total×100% |
| 平均情绪 | 情绪评分平均 | AVG(mood_score) |

#### 3.7.2 图表展示

| 图表 | 类型 | 描述 |
|------|------|------|
| 记录趋势 | 折线图 | 每日记录数量趋势 |
| 情绪分布 | 饼图 | 各情绪占比 |
| 目标进度 | 环形图 | 目标完成情况 |
| 标签云 | 标签 | 高频使用标签 |

| 错误类型 | 代码 | 说明 |

| StorageError | E002 | 存储操作错误 |
